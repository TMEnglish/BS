To clone this repository, 

1. Click on the green "Clone or Download" button on the right, above, and copy the link.

2. Click on the + (plus sign) in the top right corner of the page, and selct "Import repository."

3. Paste the link to my repository where you're prompted for a source, and then enter a name for the clone (perhaps tacking the date onto the end).

To run your clone of my repository, you'll need to register it at [MyBinder.org](https://mybinder.org/). Just copy-and-paste the URL of your clone where you're asked for a URL. They'll give you a link in response. Edit the README file of the repository, and replace the link for my binder with the link for yours.

That is all.
